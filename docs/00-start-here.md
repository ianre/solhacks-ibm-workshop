# Start Here

This workshop is designed to be completed in about ~60 minutes. You do not need to write code, but you do need to move quickly and stay on the numbered path.

## Before you begin

0. Assess if we should pair up.
Follow the instructor through:
1. Create an IBM Clouda account:
    - Go to https://cloud.ibm.com/registration?utm_content=academicsb
    - Register for a Free IBM Cloud Account
2. Provision watsonx orchestrate instance (Free Tier)
3. Go to https://www.twilio.com/try-twilio
    - Follow steps to make an account and obtain your Account ID and Auth Token

Have these ready before following the instructions in this repo:
- An IBM watsonx Orchestrate account you can log into
- A Twilio account if you want to do the WhatsApp step
- A phone with WhatsApp installed
- This repo open on your laptop

## What success looks like

By the end of the session, you should have:

1. Built two specialist agents
2. Built one supervisor agent
3. Tested the bot in the watsonx Orchestrate web chat
4. Optionally connected the bot to WhatsApp through Twilio
